The AI Landscape
Big Picture

Everything you're hearing about — AI, LLMs, Agents, Coding tools — explained clearly with diagrams and hierarchy.

01 — The Stack

The Hierarchy: How Everything Fits

Foundation Layer
🧠 AI / Machine Learning
The broad field. Teaching computers to learn from data. Everything else is built on this.
Subset of AI
📖 LLM — Large Language Models
AI models trained on massive text. They understand and generate human language. Examples: GPT-4, Claude, Gemini, Llama. The "brain" that powers most modern AI products.
Built on top of LLMs
🤖 AI Agents
An LLM given the ability to take actions — browse the web, run code, read files, call APIs. They can complete multi-step tasks autonomously. An agent = LLM + Tools + Loop.
Uses Agents
🛠️ AI Coding Tools (Cursor, Kiro, Q, etc.)
Developer-facing products. They wrap LLMs/Agents into an IDE or CLI experience so you can build software faster. The tools you use day-to-day.

04 — The Tools You've Heard Of

Every Tool Explained

⬛
Cursor
AI-Native IDE
A fork of VS Code with an LLM baked in. It understands your whole codebase and can write, refactor, and explain code inline. The most popular AI coding IDE for web/backend devs.
🌀
Kiro / Kiro CLI
AI Coding Agent (AWS)
Amazon's agentic coding assistant. Works autonomously on multi-step tasks. The CLI version lets it run in your terminal and modify files, run builds, debug — like a junior dev at your side.
🪐
Amazon Q (in Android Studio)
IDE Plugin (AWS)
Amazon Q Developer is a plugin for Android Studio (and IntelliJ). It provides inline completions, chat, code explanations, and can scan for vulnerabilities — powered by AWS Bedrock LLMs.
🔮
Antigravity
AI Coding Agent
An autonomous coding agent focused on full-feature implementation from a spec or description. Similar to Devin — it plans, codes, runs, and iterates without you driving every step.
🤖
GitHub Copilot
AI Autocomplete + Chat
The OG. Lives in your IDE. Autocompletes code line-by-line, has a chat panel, and can review PRs. Powered by OpenAI Codex and GPT-4. Also works in Android Studio via plugin.
💻
Claude Code
Terminal Agent (Anthropic)
A CLI agent that reads your whole repo, writes files, runs tests, and fixes bugs autonomously. You give it a task in natural language and it does the work. Very strong for multi-file refactors.
✨
Gemini in Android Studio
Official Google Plugin
Google's native AI integration in Android Studio (Ladybug+). Offers code completions, Jetpack Compose previews, logcat analysis, crash insights, and Studio Bot chat. Most relevant to you as an Android dev.
🌐
MCP (Model Context Protocol)
Protocol / Standard
Not a product — a standard by Anthropic. Like USB-C for AI. Lets LLMs connect to external tools (Notion, GitHub, databases) in a standardized way. Agents use MCP to access your systems.

06 — Quick Reference

Key Terms Decoded

Token A chunk of text (~4 chars). LLMs process tokens, not words. You pay per token in cloud APIs.
Context Window How much text an LLM can "see" at once. Bigger = can read more of your codebase at once.
RAG Retrieval Augmented Generation. Fetch relevant docs, inject into prompt so LLM "knows" your data without retraining.
Fine-tuning Retraining a base LLM on your specific data. Expensive but makes it an expert on your domain.
Prompt The instruction you give the LLM. Prompt engineering = writing better instructions to get better output.
Embedding A number array that represents text meaning. Used for semantic search — "find similar" rather than keyword match.
Function Calling Structured way to let an LLM call your code functions. The foundation of how agents use tools.
MCP Model Context Protocol. Standard protocol for connecting LLMs to external tools/data sources. Like REST but for AI agents.
Inference Running the model to generate output. "Running inference on device" = on-device AI, no cloud call needed.
Hallucination When LLMs confidently generate false information. A core challenge — always verify AI output for critical data.
System Prompt Hidden instructions given to the LLM before the user message. Sets personality, rules, and constraints.
Agentic Describes AI that acts autonomously over multiple steps, using tools, rather than just responding to one message.
